ReactJs Client Project (International Tourist & Travel Website)

BlueHawk is a responsive tourist and travel website built with React.It fetches travel data from external APIs and provides an engaging, mobile-friendly UI responsive with animations and contact integration. It was built as a OneKey Consultancy Client project and deployed on Firebase and the client offical website.


Tech-Stack: HTML, Tailwind CSS, Javascript, React, Bootstrap,AI tool(chatgpt, gemini, github-copilot, claude, google ai studio) etc. 


It is a Frontend Project in which APIs key used and for designing advance Tailwind CSS. In this project different type of google font is used
